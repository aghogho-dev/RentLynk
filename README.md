## Prerequisites

- **Node.js** v18 or newer  
- **npm** (bundled with Node.js) or **Yarn**

---

## Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/<your-org>/rentlynk.git
cd rentlynk
```

### 2. Install dependencies
```bash
npm install
# or
yarn install
```


### 3. (Optional) Initialize Tailwind CSS

If you havenâ€™t yet generated a `tailwind.config.js`, run:

```bash
npm run init:tailwind
# or
yarn init:tailwind
```


---

## Running Locally

Start the Next.js dev server with Turbopack:

```bash
npm run dev
# or
yarn dev
```

Open http://localhost:3000 in your browser.
